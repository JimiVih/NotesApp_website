Node versio:
Node.js v18.18.0

'npm install' terminaaliin niin projektiin asennetaan tarvittavat lisäosat
'npm run dev' terminaaliin niin projekti käynnistyy


NotesApp iconista pääsee takaisin valikkoon

create new note nappi vie "/newNote" sivulle
sivulle pääsee vasta kun on luonut kurssin
sivulla täytyy valita kurssi, lisätä vähintään yksi merkki ja paina save, kun on valmis

create new course nappi vie "/newCourse" sivulle
sivulla täytyy lisätä vähintään yksi merkki ja paina save, kun on valmis

list notes nappi vie "/lisNotes" sivulle, jossa voi valita haluaako nähdä kaikki muistilaput vai tietyn jakson muistilaput
muistilapun voi poistaa painamalla "X" nappia muisilapun oikeassa yläreunassa
